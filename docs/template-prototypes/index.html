How to shortlist from 15

You don’t need fifteen production templates. Use this set as a menu of patterns, then ship 2–4 catalog entries:

  1. Portal Split (01) — default second sales template (broadest fit).
  2. Luxury Editorial (02) or Private Client (07) — one premium skin for brand-led firms.
  3. Lettings Practical (12) — department-specific when lettings is active.
  4. Bento Modular (10) — not necessarily a public template name, but a great implementation model for optional modules.

Steal freely across directions: e.g. Portal Split structure + Soft Scandi colour, or Spec Atelier docs + Agent Spotlight trust.
